<p><b>Объединённая таксономи́ческая информацио́нная слу́жба</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&#160;<span lang="en" style="font-style:italic;">Integráted Taxonómic Informátion Sýstem</span>, <b>ITIS</b>)&#160;— межправительственное партнёрство с участием <a href="/wiki/%D0%A1%D0%A8%D0%90" class="mw-redirect" title="США">США</a>, <a href="/wiki/%D0%9A%D0%B0%D0%BD%D0%B0%D0%B4%D0%B0" title="Канада">Канады</a> и <a href="/wiki/%D0%9C%D0%B5%D0%BA%D1%81%D0%B8%D0%BA%D0%B0" title="Мексика">Мексики</a>, а также других организаций и специалистов, имеющее целью создание и поддержание полной и всесторонней открытой <a href="/wiki/%D0%A2%D0%B0%D0%BA%D1%81%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%8F" title="Таксономия">таксономической</a> базы данных по всем биологическим <a href="/wiki/%D0%92%D0%B8%D0%B4_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" class="mw-redirect" title="Вид (биология)">видам</a>.
</p><p>Эта служба была создана в середине 1990-х годов при поддержке нескольких федеральных агентств США, в том числе <a href="/wiki/%D0%90%D0%B3%D0%B5%D0%BD%D1%82%D1%81%D1%82%D0%B2%D0%BE_%D0%BF%D0%BE_%D0%BE%D1%85%D1%80%D0%B0%D0%BD%D0%B5_%D0%BE%D0%BA%D1%80%D1%83%D0%B6%D0%B0%D1%8E%D1%89%D0%B5%D0%B9_%D1%81%D1%80%D0%B5%D0%B4%D1%8B_%D0%A1%D0%A8%D0%90" title="Агентство по охране окружающей среды США">Агентства по охране окружающей среды</a>, <a href="/wiki/%D0%9C%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D0%B5%D1%80%D1%81%D1%82%D0%B2%D0%BE_%D0%B2%D0%BD%D1%83%D1%82%D1%80%D0%B5%D0%BD%D0%BD%D0%B8%D1%85_%D0%B4%D0%B5%D0%BB_%D0%A1%D0%A8%D0%90" title="Министерство внутренних дел США">Министерства внутренних дел</a> и <a href="/wiki/%D0%9D%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BC%D1%83%D0%B7%D0%B5%D0%B9_%D0%B5%D1%81%D1%82%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%BE%D0%B9_%D0%B8%D1%81%D1%82%D0%BE%D1%80%D0%B8%D0%B8_(%D0%92%D0%B0%D1%88%D0%B8%D0%BD%D0%B3%D1%82%D0%BE%D0%BD)" title="Национальный музей естественной истории (Вашингтон)">Национального музея естественной истории</a><sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup><sup id="cite_ref-:0_3-0" class="reference"><a href="#cite_note-:0-3">&#91;3&#93;</a></sup> Первоначальной задачей проекта было предоставление таксономической информации для таксонов <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0" title="Северная Америка">Северной Америки</a>. В базе данных ITIS каждому имени, независимо от таксономического статуса или положения в иерархии, присваивается уникальный идентификатор (ID). Схема базы данных полностью документирована. Вся база данных доступна для загрузки по <a href="/wiki/FTP" title="FTP">FTP</a> в виде схемы <a href="/wiki/SQL" title="SQL">SQL</a> с данными в текстовых файлах с разделителями. Это позволяет использовать данные ITIS в качестве источника таксономических данных в проектах по информатике биоразнообразия<sup id="cite_ref-:0_3-1" class="reference"><a href="#cite_note-:0-3">&#91;3&#93;</a></sup>. К 2017 году эта база данных содержала более 840 000 научных названий<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup>.
</p>
