<h2><span id=".D0.A1.D0.BE.D0.B1.D1.8B.D1.82.D0.B8.D1.8F"></span><span class="mw-headline" id="События">События</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1797_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BD%D0%B0%D1%83%D0%BA%D0%B5&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Редактировать раздел «События»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=1797_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BD%D0%B0%D1%83%D0%BA%D0%B5&amp;action=edit&amp;section=1" title="Редактировать раздел «События»">править код</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li><a href="/wiki/%D0%9A%D0%B0%D0%B2%D0%B5%D0%BD%D0%B4%D0%B8%D1%88,_%D0%93%D0%B5%D0%BD%D1%80%D0%B8" title="Кавендиш, Генри">Генри Кавендишем</a> осуществлено первое экспериментальное измерение <a href="/wiki/%D0%93%D1%80%D0%B0%D0%B2%D0%B8%D1%82%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D1%8F%D0%BD%D0%BD%D0%B0%D1%8F" title="Гравитационная постоянная">гравитационной постоянной</a>, названное <a href="/wiki/%D0%AD%D0%BA%D1%81%D0%BF%D0%B5%D1%80%D0%B8%D0%BC%D0%B5%D0%BD%D1%82_%D0%9A%D0%B0%D0%B2%D0%B5%D0%BD%D0%B4%D0%B8%D1%88%D0%B0" title="Эксперимент Кавендиша">экспериментом Кавендиша</a>.</li>
<li><a href="/wiki/%D0%A2%D0%B5%D0%BD%D0%BD%D0%B0%D0%BD%D1%82,_%D0%A1%D0%BC%D0%B8%D1%82%D1%81%D0%BE%D0%BD" title="Теннант, Смитсон">Смитсон Теннант</a> приводит доказательства, что <a href="/wiki/%D0%91%D1%80%D0%B8%D0%BB%D0%BB%D0%B8%D0%B0%D0%BD%D1%82" title="Бриллиант">бриллиант</a> состоит из чистого <a href="/wiki/%D0%A3%D0%B3%D0%BB%D0%B5%D1%80%D0%BE%D0%B4" title="Углерод">углерода</a>.</li>
<li><a href="/wiki/%D0%92%D0%BE%D0%BA%D0%BB%D0%B5%D0%BD,_%D0%9B%D1%83%D0%B8_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0" title="Воклен, Луи Никола">Луи Воклен</a> выделил из «сибирского красного свинца» (PbCrO<small>4</small>, современное название&#160;— крокоит) новый тугоплавкий металл (скорее всего был получен карбид хрома).</li>
<li><a href="/wiki/%D0%9F%D1%80%D1%83%D1%81%D1%82,_%D0%96%D0%BE%D0%B7%D0%B5%D1%84_%D0%9B%D1%83%D0%B8" title="Пруст, Жозеф Луи">Жозеф Луи Пруст</a> начал исследования ставшие основой для открытия им <a href="/wiki/%D0%97%D0%B0%D0%BA%D0%BE%D0%BD_%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D1%8F%D0%BD%D1%81%D1%82%D0%B2%D0%B0_%D1%81%D0%BE%D1%81%D1%82%D0%B0%D0%B2%D0%B0" title="Закон постоянства состава">закона постоянства состава</a>.</li>
<li>Закрыт <a href="/wiki/%D0%A1%D1%82%D0%B0%D1%80%D1%8B%D0%B9_%D0%9B%D1%91%D0%B2%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82" title="Старый Лёвенский университет">Старый Лёвенский (<i>устар.</i> лувенский) университет</a>&#160;— существовавший в городе <a href="/wiki/%D0%9B%D1%91%D0%B2%D0%B5%D0%BD" title="Лёвен">Лёвене</a> (на тот момент <a href="/wiki/%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3%D1%81%D1%82%D0%B2%D0%BE)" title="Брабант (герцогство)">герцогство Брабант</a>, <a href="/wiki/%D0%91%D1%83%D1%80%D0%B3%D1%83%D0%BD%D0%B4%D1%81%D0%BA%D0%B8%D0%B5_%D0%9D%D0%B8%D0%B4%D0%B5%D1%80%D0%BB%D0%B0%D0%BD%D0%B4%D1%8B" title="Бургундские Нидерланды">Бургундские Нидерланды</a>; ныне <a href="/wiki/%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F" title="Бельгия">Бельгия</a>) с 1425 по 1797 годы университет, считавшийся до XVI века лучшим в Европе<sup id="cite_ref-ЭСБЕ_1-0" class="reference"><a href="#cite_note-ЭСБЕ-1">&#91;1&#93;</a></sup>.</li>
<li>Основана <a href="/wiki/%D0%90%D0%BA%D0%B0%D0%B4%D0%B5%D0%BC%D0%B8%D1%8F_%D0%A1%D1%91%D1%85%D1%8D%D0%B9%D0%B4%D0%B7%D0%B0%D0%BA%D0%B0" title="Академия Сёхэйдзака">Академия Сёхэйдзака</a>.</li>
