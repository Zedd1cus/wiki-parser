	<h1 id="firstHeading" class="firstHeading mw-first-heading">Bibcode</h1>
	<div id="bodyContent" class="vector-body">

		<div id="mw-content-text" class="mw-body-content mw-content-ltr" lang="ru" dir="ltr"><div class="mw-parser-output"><dl><dd><span class="plainlinks selfreference"><i>Существует шаблон Википедии, предназначенный для ссылок по идентификатору bibcode, см. <style data-mw-deduplicate="TemplateStyles:r117895138">.mw-parser-output .ts-templateCallCode-weak{color:#72777d}.mw-parser-output .ts-templateCallCode-pipe{margin:0 2px}.mw-parser-output .ts-templateCallCode-pipe+.ts-templateCallCode-pipe,.mw-parser-output .ts-templateCallCode-pipe+.ts-templateCallCode-param>.ts-templateCallCode-weak:first-child>.ts-templateCallCode-pipe:first-child{margin-left:0}.mw-parser-output .ts-templateCallCode-param+.ts-templateCallCode-closing{margin-left:2px}.mw-parser-output span.ts-templateCallCode>.ts-templateCallCode-templateName a{padding:0 0.5em!important;position:relative;margin:-0.5em}</style><span class="ts-templateCallCode"><span class="ts-templateCallCode-opening ts-templateCallCode-weak">{{</span><span class="ts-templateCallCode-templateName" data-navboxnavigation-link="0"><a href="/wiki/%D0%A8%D0%B0%D0%B1%D0%BB%D0%BE%D0%BD:Bibcode" title="Шаблон:Bibcode">bibcode</a></span><span class="ts-templateCallCode-closing ts-templateCallCode-weak">}}</span></span>.</i></span></dd></dl>
<p><b>Bibcode</b>&#160;— <a href="/wiki/%D0%98%D0%B4%D0%B5%D0%BD%D1%82%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%82%D0%BE%D1%80" title="Идентификатор">идентификатор</a>, используемый многими <a href="/wiki/%D0%90%D1%81%D1%82%D1%80%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%8F" title="Астрономия">астрономическими</a> базами данных и каталогами для ссылок на литературу. Bibcode был разработан для использования в <a href="/wiki/SIMBAD" title="SIMBAD">SIMBAD</a> и базе данных о внегалактических объектах <span data-interwiki-lang="en" data-interwiki-article="NASA/IPAC Extragalactic Database"><a href="/w/index.php?title=NASA/IPAC_Extragalactic_Database&amp;action=edit&amp;redlink=1" class="new" title="NASA/IPAC Extragalactic Database (страница отсутствует)">NASA/IPAC Extragalactic Database</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a href="https://en.wikipedia.org/wiki/NASA/IPAC_Extragalactic_Database" class="extiw" title="en:NASA/IPAC Extragalactic Database"><span title="NASA/IPAC Extragalactic Database — версия статьи «NASA/IPAC Extragalactic Database» на английском языке">[en]</span></a></sup></span> (NED), но в настоящее время используется более широко, например, в ADS (<a href="/wiki/%D0%90%D1%81%D1%82%D1%80%D0%BE%D1%84%D0%B8%D0%B7%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BD%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D0%9D%D0%90%D0%A1%D0%90" title="Астрофизическая информационная система НАСА">Астрофизическая информационная система НАСА</a>)<sup id="cite_ref-a_1-0" class="reference"><a href="#cite_note-a-1">&#91;1&#93;</a></sup><sup id="cite_ref-b_2-0" class="reference"><a href="#cite_note-b-2">&#91;2&#93;</a></sup>. Код состоит ровно из 19 символов и имеет форму
</p>
<center><code>YYYYJJJJJVVVVMPPPPA</code> ,</center>
<p>где YYYY&#160;— 4 цифры года публикации, JJJJJ&#160;— код журнала/монографии/сборника. В случае ссылки на журнал, VVVV&#160;— номер тома, M означает буквенный индекс раздела журнала (например, L для секции Letters), PPPP&#160;— номер начальной страницы, A&#160;— первая буква фамилии первого автора. Точки (.) используются для заполнения неиспользуемых полей, чтобы длина каждого поля всегда была равна вышеуказанной; точки добавляются справа для кода JJJJJ и слева для номера тома и страницы<sup id="cite_ref-a_1-1" class="reference"><a href="#cite_note-a-1">&#91;1&#93;</a></sup><sup id="cite_ref-b_2-1" class="reference"><a href="#cite_note-b-2">&#91;2&#93;</a></sup>.
